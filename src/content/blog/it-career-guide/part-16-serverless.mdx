## 2. AWS Lambda Fundamentals

### Function Handler

```python
import json

def lambda_handler(event, context):
    # Parse event
    http_method = event['httpMethod']
    path = event['path']
    
    # Business logic
    if http_method == 'GET' and path == '/users':
        return {
            'statusCode': 200,
            'headers': {
                'Content-Type': 'application/json',
                'Access-Control-Allow-Origin': '*'
            },
            'body': json.dumps({'users': []})
        }
    
    return {
        'statusCode': 404,
        'body': json.dumps({'error': 'Not found'})
    }
```

### Environment Variables

```python
import os
import boto3

# Access environment variables
DB_HOST = os.environ['DB_HOST']
DB_PORT = os.environ.get('DB_PORT', '5432')

# AWS SDK clients

    # Use configuration
    table = dynamodb.Table(os.environ['TABLE_NAME'])
    return table.scan()
```

## 3. Cloudflare Workers

### Worker Script

```javascript
export default {
  async fetch(request, env, ctx) {
    const url = new URL(request.url);
    
    // Route handling
    if (url.pathname === '/api/users') {
      return handleUsers(request, env);
    }
    
    if (url.pathname === '/api/orders') {
      return handleOrders(request, env);
    }
    
    return new Response('Not Found', { status: 404 });
  }
};

async function handleUsers(request, env) {
  if (request.method === 'GET') {
    const users = await env.DB.prepare('SELECT * FROM users').all();
    return new Response(JSON.stringify(users), {
      headers: { 'Content-Type': 'application/json' }
    });
  }
  
  if (request.method === 'POST') {
    const body = await request.json();
    const result = await env.DB.prepare(
      'INSERT INTO users (name, email) VALUES (?, ?)'
    ).bind(body.name, body.email).run();
    
    return new Response(JSON.stringify({ id: result.lastRowid }), {
      status: 201
    });
  }
}
```

### KV Storage

```javascript
export default {
  async fetch(request, env) {
    const url = new URL(request.url);
    
    // PUT - Store data
    if (url.pathname === '/cache' && request.method === 'PUT') {
      const data = await request.text();
      await env.MY_KV.put('my-key', data);
      return new Response('Stored');
    }
    
    // GET - Retrieve data
    if (url.pathname === '/cache' && request.method === 'GET') {
      const data = await env.MY_KV.get('my-key');
      return new Response(data || 'Not found', {
        headers: { 'Content-Type': 'text/plain' }
      });
    }
  }
};
```

### D1 Database (SQLite)

```javascript
    ).bind(1).first();
    
    return new Response(JSON.stringify(result), {
      headers: { 'Content-Type': 'application/json' }
    });
  }
};
```

## 4. Cold Start Optimization

### AWS Lambda Optimization

```python
# Initialize outside handler for reuse
import boto3
import json
from typing import Any

# Global variables reused across invocations
S3_CLIENT = boto3.client('s3')
DYNAMODB_TABLE = boto3.resource('dynamodb').Table('MyTable')

# Pre-warmed connections
DB_CONNECTION = None

def get_db_connection():
    global DB_CONNECTION
    if DB_CONNECTION is None:
        DB_CONNECTION = establish_connection()
    return DB_CONNECTION

def lambda_handler(event: dict, context: Any) -> dict:
    # Use pre-initialized resources
    db = get_db_connection()
    return process_request(event, db)
```

### Cloudflare Workers Optimization

### Global Deployment

```javascript
[[routes]]
pattern = "api.example.com/*"

# Edge functions run in 300+ locations worldwide
```

### Data Localization

```javascript
export default {
  async fetch(request, env) {
    const url = new URL(request.url);
    const region = getRegionFromPath(url.pathname);
    
    // Route to appropriate regional database
    const db = getDatabaseForRegion(region);
    const data = await db.query('SELECT * FROM users');
    
    return new Response(JSON.stringify(data));
  }
};

function getRegionFromPath(path) {
  const regions = {
    '/us/': 'us-east-1',
    '/eu/': 'eu-west-1',
    '/ap/': 'ap-southeast-1'
  };
  
  for (const [prefix, region] of Object.entries(regions)) {
    if (path.startsWith(prefix)) return region;
  }
  return 'us-east-1';
}
```

## 6. Serverless API Gateway

### REST API with AWS API Gateway

```yaml
# AWS SAM template
Resources:
  MyApi:
    Type: AWS::Serverless::Api
    Properties:
      StageName: Prod
      DefinitionBody:
        swagger: "2.0"
        info:
          title: My API
        paths:
          /users:
            get:
              x-amazon-apigateway-integration:
                httpMethod: POST
                type: aws_proxy
                uri: !Sub "arn:aws:apigateway:${AWS::Region}:lambda:path/2015-03-31/functions/${MyFunction.Arn}/invocations"
```

### HTTP API (Faster, Cheaper)

```python
# AWS SAM template for HTTP API
Resources:
  MyHttpApi:
    Type: AWS::Serverless::HttpApi
    Properties:
      RouteConfig:
        Path: /users
        Method: get
        IntegrationUri: !GetAtt MyFunction.Arn
```

## 7. Serverless Databases

### DynamoDB

```python
import boto3
from boto3.dynamodb.conditions import Key

dynamodb = boto3.resource('dynamodb')
table = dynamodb.Table('Users')

def get_user(user_id):
    response = table.get_item(Key={'id': user_id})
    return response.get('Item')

def query_users_by_email(email_prefix):
    response = table.query(
        IndexName='EmailIndex',
        KeyConditionExpression=Key('email').begins_with(email_prefix)
    )
    return response['Items']
```

### Supabase (PostgreSQL)

```python
from supabase import create_client
import os

supabase = create_client(
    os.environ['SUPABASE_URL'],
    os.environ['SUPABASE_KEY']
)

def get_users():
    response = supabase.table('users').select('*').execute()
    return response.data
```

### Turso (LibSQL)

```python
import libsql

conn = libsql.connect(database="turso://your-db.turso.io", auth_token="your-token")

def get_users():
    result = conn.execute("SELECT * FROM users")
    return result.fetchall()
```

## 9. Common Serverless Interview Questions

| Question | Answer |
|----------|--------|
| **What are the benefits of serverless?** | Auto-scaling, cost efficiency, no server management, faster deployment. |
| **How to handle state in serverless?** | Use external databases (DynamoDB, RDS, Redis) or storage services. |
| **What are cold starts?** | Delay when serverless function hasn't been invoked recently. Mitigate with provisioned concurrency. |
| **How to secure serverless functions?** | IAM roles, API keys, JWT, resource policies. |
| **What is the difference between Lambda and Cloudflare Workers?** | Lambda is AWS-specific, Workers are edge computing with global deployment. |
